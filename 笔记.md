### 2. Skill 完善

**目标效果：** 首次发布时，已有 Agent 的用户只需安装我们的 Skill，即可创建 Slides。创建出来的 Slides 需符合我们的 Contract 格式，并在创建成功后自动打开编辑器，方便用户手动编辑。

**优先级策略：** 当前 Agent 通常会安装大量 Skill。如果用户的 Agent 中同时安装了其他类似的 PPT 类 Skill，我们希望通过优化 Skill 的 Prompt 公式，将自己的 Skill 优先级提到最高，从而在"兼容"层面实现领先。

**反馈功能：** 计划在 Skill 中加入反馈机制——留一个口子，让用户 Agent 在遇到问题或无法解决的事项时，能通过一个简单的工具将反馈发送回来。具体实现方案待研究。
